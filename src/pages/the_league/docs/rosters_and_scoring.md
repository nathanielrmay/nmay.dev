# Rosters & Scoring

## Roster Construction

Rosters include 9 scoring participants per league:

| NFL    | NBA    | MLB              | MLS    |
| ------ | ------ | ---------------- | ------ |
| 1 QB   | 2 G    | 1 C              | 2 F    |
| 1 RB   | 2 F    | 1 Corner IF (CI) | 2 MF   |
| 3 WR   | 1 C    | 1 Middle IF (MI) | 2 D    |
| 1 TE   |        | 2 OF             | 1 G    |
| 1 K    |        | 3 P              |        |
| 2 Flex | 4 Flex |                  | 1 Flex |
| 1 Team | 1 Team | 1 Team           | 1 Team |

From the activity calendar in the *Draft, Waivers & Game Loop* document, we can see that October requires filling 4 active fantasy rosters (NFL, NBA, MLB, MLS). This is the month when roster spots are the most valuable. With 4 rosters of 9 players each, the minimum number of roster spots needed would be 36. **Total team roster size is 40.**

There is no injured reserve list — an injured player consumes a roster spot. This is intentional; it encourages active roster management and rewards owners who stay engaged with player news rather than stashing injured talent on a protected list.

## Scoring

Each sport is scored in rotisserie style with 5 player categories and 1 team category, except for College Football and College Basketball which only compete in the team wins category.

Assuming 10 teams in the league: 4 sports × 5 player categories = 20 roto categories, plus 1 combined team wins category = **21 total categories**. With a max of 10 points per category, the **maximum possible score at the end of the year is 210**.

| NFL           | NBA                     | MLB         | MLS            | CFB       | CBB       |
| ------------- | ----------------------- | ----------- | -------------- | --------- | --------- |
| FPPW          | FPPW                    | Total Bases | FPPW           |           |           |
| Passing Yards | Rebounds + Assists (RA) | Slugging    | Goals          |           |           |
| AP Yards      | Blocks + Steals (BS)    | Wins        | Passing %      |           |           |
| AP TDs        | eFG%                    | ERA*        | Minutes Played |           |           |
| Points Scored | +/−                     | WAR         | Saves          |           |           |
| Team Wins     | Team Wins               | Team Wins   | Team Wins      | Team Wins | Team Wins |

*\*ERA is an inverted category — lower is better.*

## Team Wins Scoring

All team wins contribute to a single combined "wins" category, weighted by league:

| League | Win Value                |
| ------ | ------------------------ |
| NFL    | 1.0                     |
| NBA    | 0.2                     |
| MLB    | 0.1                     |
| MLS    | 0.5                     |
| CFB    | In-conference: 1.0      |
|        | Out-of-conference: 0.25 |
|        | Bowl / Playoff: 2.0     |
| CBB    | In-conference: 0.5      |
|        | Out-of-conference: 0.25 |
|        | NCAA Tournament: 1.0    |
